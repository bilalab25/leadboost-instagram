💡 PROMPT: Actualiza mi componente `IntegrationsTab.tsx` para que soporte conexiones a Facebook, Instagram, Threads y WhatsApp

### 🎯 Contexto

Actualmente mi componente `IntegrationsTab.tsx` ya maneja Facebook e Instagram manualmente con `handleConnectFacebook` y `handleConnectInstagram`.  
Quiero simplificarlo para usar un **solo manejador genérico `handleConnect(provider)`** y agregar soporte para **Threads** y **WhatsApp** sin duplicar lógica.

---

### 🧱 REQUISITOS

1. Mantén el mismo diseño y estructura del componente.  
   (usa Tailwind, Dialog, Accordion, etc. como está en mi versión actual)

2. Reemplaza los métodos:
   ```ts
   handleConnectFacebook()
   handleConnectInstagram()
   ```
   por un único método:

   ```ts
   const handleConnect = (provider: string) => {
     let url = "";

     if (["facebook", "instagram", "threads"].includes(provider)) {
       // 🔁 Todas usan el flujo OAuth de Facebook
       url = "/api/integrations/facebook/connect";
     } else if (provider === "whatsapp") {
       // 🔁 WhatsApp usa un POST directo
       fetch("/api/integrations/whatsapp/connect", { method: "POST" })
         .then(res => res.json())
         .then(() => window.location.reload())
         .catch(err => console.error("WhatsApp connect error:", err));
       return;
     }

     // 🔄 Para OAuth (Facebook/Instagram/Threads)
     const popup = window.open(url, "_blank", "width=600,height=700");

     const timer = setInterval(() => {
       if (popup?.closed) {
         clearInterval(timer);
         window.location.reload();
       }
     }, 1000);
   };
   ```
